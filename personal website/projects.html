IoT Remote Control Web Application

A collaborative university project to create a secure web application for remotely managing and monitoring various IoT devices. The system provided a centralized dashboard to view a live camera feed, adjust a smart thermostat, and control a remote door lock, enhancing smart home management capabilities. Due to privacy concerns we are unable to show the live feed in the demo, the Home Assistant section of the webpage is to view the live camera feed once logged in. This project would also be able to use other IoT devices that use APIs, like Philips Hue series lights, light sensors, and alarm systems.
