Insurlytics - Insurance Analytics Dashboard

Overview

Insurlytics is a data visualization tool designed to help insurance firms make data-driven decisions. By aggregating complex datasets into intuitive charts and graphs, the dashboard provides clear visibility into key performance indicators (KPIs) such as policy claims, revenue distribution, and risk assessment.

Tech Stack

HTML5 CSS3 JavaScript Chart.js

Key Features

  • Interactive Data Visualization: Dynamic charts (Line, Bar, Doughnut) powered by Chart.js.
  • Data Filtering: Real-time filtering by date range, policy type, and region.
  • Responsive Tables: Sortable and paginated tables for detailed data inspection.
  • Modern UI: Clean, dashboard-style layout optimized for quick readability.

Challenges & Solutions (STAR)

Situation: The client was struggling to identify profitability trends because their data was scattered across multiple static spreadsheets.

Task: Create a centralized, visual dashboard that could reveal trends and outliers in the insurance data at a glance.

Action: I utilized Chart.js to build a responsive frontend application. I focused on designing a color-coded visual hierarchy that immediately drew attention to critical metrics like high-claim ratios.

Result: The new dashboard allowed stakeholders to identify high-risk policy segments 50% faster, leading to more informed underwriting strategies.
